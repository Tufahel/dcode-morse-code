# Morse Code

> This project is about decoding a morse code using ruby.

## Built With

- Ruby
- Terminal
- IRB
- VSCODE

## Getting Started

- Clone this repo `https://github.com/Tufahel/dcode-morse-code.git`
- Use command `cd dcode-morse-code` and have fun.
